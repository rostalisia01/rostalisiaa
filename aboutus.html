Tentang Saya

Nama saya Rosta lisia saya lahir di nangahpinoh, 4 juli 2000 . saya tinggal di 28 oktober Gg arhas permai jalur 2 no 28 kalimantan barat jenis kelamin saya wanita. Saya beragama katolik.

Pekerjaan saya saat ini adalah Sebagai Mahasiswa di Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( STIE Indonesia Pontianak ) dengan program studi S1 Manajemen.

Saya anak pertama dari tiga bersaudara. Ayah saya bernama Widianto dan ibu saya bernama Anarusmini.

Riwayat pendidikan :

- SD Negeri 2 jungkat ( Dari Tahun 2006 - Tahun 2012 )

- SMP Khatulistiwa jungkat ( Dari Tahun 2012 - Tahun 2015 )

- SMK Imanuel II sungairaya ( Dari Tahun 2015 - 2019 )

- Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( Dari Tahun 2019 - Sekarang )
